# Responsive Personal Portfolio (Professional Version)

A fully responsive portfolio webpage crafted with pure HTML and CSS.

## Features

- Semantic HTML5 structure, accessible and keyboard-friendly
- Custom animated hamburger menu for mobile
- Colorful, modern design with blue-green/gold theme
- Skills section in responsive grid (3/2/1-col layout)
- Polished hover and transition effects throughout
- No frameworks, no JS libraries
- About and Contact sections for depth
- Social media links footer

## AI Tool Used

- Initial structure was bootstrapped with bolt.new
- All CSS, accessibility, layout, animation, and color improvements were manually implemented.

## Manual Improvements

- Media queries for professional responsiveness
- Accessibility/ARIA for navigation/menu
- Color, typography, and UI polish
- Refactored CSS organization and variable usage
- Added subtle animations and transitions
